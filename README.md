# Проверка сбалансированности скобок

Проект реализует структуру данных **Стек (Stack)** и использует её для проверки корректности расстановки скобок в строке. Алгоритм определяет, является ли последовательность скобок сбалансированной (каждой открывающей скобке соответствует закрывающая, и они правильно вложены).

## Возможности

- **Класс `Stack`**: Полная реализация стека с методами:
  - `is_empty()` — проверка на пустоту.
  - `push(item)` — добавление элемента.
  - `pop()` — удаление и возврат верхнего элемента.
  - `peek()` — просмотр верхнего элемента без удаления.
  - `size()` — получение размера стека.
- **Проверка баланса**: Поддержка трех типов скобок: `()`, `[]`, `{}`.
- **Валидация**: Мгновенное определение несбалансированных последовательностей.

## Структура проекта

- `stack_lifo.py` — реализация класса `Stack`.
- `brackets_check.py` — функция проверки сбалансированности и точка входа программы.

## Запуск

Проект написан на **Python 3** и не требует внешних зависимостей.

Убедитесь, что оба файла находятся в одной директории, затем выполните:

```bash
python brackets_check.py
```
